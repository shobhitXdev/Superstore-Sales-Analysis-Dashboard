# Superstore Sales Analysis Dashboard (Excel)

## Project Overview
This project involved analyzing a retail dataset to extract actionable insights. I built an interactive dashboard in Excel to track key performance metrics and visualize sales trends across different regions and categories.

## Key Features
- **Interactive KPIs:** Total Sales, Total Profit, Quantity, and Profit Margin.
- **Trend Analysis:** Yearly sales growth tracking.
- **Regional Performance:** Distribution of profit across different geographical regions.
- **Product Insights:** Identification of top-performing sub-categories.
- **Dynamic Filtering:** Slicers for Region, Category, and Segment for deep-dive analysis.

## Tools Used
- **Microsoft Excel:** Pivot Tables, Pivot Charts, Slicers, Data Cleaning (Power Query).
